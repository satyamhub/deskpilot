# DeskPilot 🚀

DeskPilot is an AI-powered computer autopilot that can perform tasks on your system and browser using natural language commands.

## ✨ Features

* Understands user commands using LLM (LLaMA 3 via Ollama)
* Breaks tasks into steps
* Automates browser actions (Playwright)
* Executes actions like a human

## 🧠 Example Commands

* "Open YouTube and search DSA problems"
* "Search best laptops under 1 lakh"
* "Go to Google and type hello"

## ⚙️ Tech Stack

* Python
* Ollama (LLaMA 3)
* Playwright
* PyAutoGUI (coming soon)

## 🚀 Getting Started

```bash
pip install -r requirements.txt
playwright install
ollama run llama3
python main.py
```
